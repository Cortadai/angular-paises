**PaisesApp** es una aplicación web desarrollada en Angular 14 que funciona como un buscador interactivo de información sobre países del mundo, consumiendo la API REST Countries v2. La aplicación permite a los usuarios explorar datos detallados de países a través de tres métodos de búsqueda distintos: por nombre de país (con autocompletado optimizado mediante debounce de 300ms), por capital, o navegando por regiones geográficas (África, América, Asia, Europa, Oceanía). Cada país puede visualizarse en detalle mostrando su bandera, población, códigos identificadores, capital y traducciones del nombre en varios idiomas. El proyecto está bien estructurado siguiendo las mejores prácticas de Angular, con módulos separados (PaisModule y SharedModule), componentes reutilizables (PaisInputComponent y PaisTablaComponent), un servicio centralizado para las llamadas HTTP (PaisService), y hace uso efectivo de RxJS con operadores como `debounceTime` y `switchMap` para optimizar las peticiones al API. La interfaz es responsive gracias a Bootstrap 5.2 e incluye animaciones con Animate.css para una mejor experiencia de usuario.
